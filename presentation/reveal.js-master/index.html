What is TypeScript
  • Developed in Microsoft
  • Was created in October 2012 Anders Hejlsberg
  • Strict syntactical superset of JavaScript
  • OpenSource project
Main Features
Type annotations and compile-time type checking
  • boolean
  • number
  • bigint
  • string
  • Object
  • Any
  • Symbol
  • null and undefined
  • unknown
  • never

Type inference

Enumerated types


								enum Color {
									Red,
									Green,
									Blue,
								  }
								  let c: Color = Color.Green;

Interfaces


								interface IUser {
									id: number;
									name: string;
								}

Tuples


								let typle: [string, number];
								typle = ["hello", 10]; // OK
								typle = [10, "hello"]; // Error

Generics


								function identity(arg: Type): Type {
									return arg;
								  }

Namespaces


								namespace Personnel {
									export class Employee {

										constructor(public name: string){
										}
									}
								}
								let alice = new Personnel.Employee("Alice");
								console.log(alice.name);    // Alice

Async/await


								async function printDelayed(elements: string[]) {
									for (const element of elements) {
									  await delay(400);
									  console.log(element);
									}
								  }

The following features are backported from ECMAScript 2015

Classes


								class User {

									name: string;
									age: number;
									print(){
										console.log(`name: ${this.name}  age: ${this.age}`);
									}
									toString(): string{
										return `${this.name}: ${this.age}`;
									}
								}

Modules

								  }

Abbreviated "arrow" syntax for anonymous functions


								let sum = (x: number, y: number) => x + y;
								let result = sum(15, 35); // 50
								console.log(result);
